## Rocket Fuel Advertising Analysis

This project analyzes A/B test data to measure the causal impact of a digital advertising campaign. It estimates conversion lift using OLS regression and calculates incremental revenue, campaign cost, ROI, and opportunity cost. The analysis also examines conversion rates by impression frequency, day, and hour.
